Basic SCSS Template
by Duncan Snider

The SCSS source directory is called scss and is located in the root directory

The directory served by the server is called public and is located in the root directory

The file the SCSS source is compiled to is called style.css and is located in /public/css

Server Commands

start

Starts the development server and watches for SCSS changes

scss

Compiles SCSS into CSS

scss:watch

Compiles SCSS into CSS and watches for changes

server

Starts the development server
